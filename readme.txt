Adds 30+ custom CSS animations and scroll/hover controls to Elementor elements.

== Description ==

Animatrix for Elementor adds a set of custom entrance, hover, and scroll animations to Elementor widgets, columns and sections. Use the controls in the Advanced tab to pick entrance/hover/scroll animations, set durations and delays.

== Installation ==

1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins screen in WordPress.
3. Open any Elementor widget, column or section and go to the Advanced tab to find the Animatrix controls.
